Intro to Programming notes

  • HTML- Hyper Text Markup Language, this is the main document type of the web, Glues everything together. HTML is made up of Text content this is what you see and Markup this is what it looks like. It also makes references to other documents, images and videos.
  • URLS- How you refer to documents on the web.
  • HTTP- protocol of the web
  • Hyperlinks- links in between pages
  • Markup is made up of tags, tags are always contained within angled brackets most having opening and closing however some that are called void tags do not require closing tags.
  • Opening Tag Content Closing tag
    • Container Elements- Inline are span, a br, img and strong. Block is div, p, form,
